Java Problem Repository and Education Platform

Welcome to the DeltaTech company website. DeltaTech is working on a very useful project for the Siena College computer science faculty. Java Problem Repository and Education Platform, or JPREP for short, is a web application that allows individuals to login to their accounts and access course assignments that were given to them by professors.
